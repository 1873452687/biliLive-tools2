# 0.8.0

## Breaking change

webhook配置黑名单支持通配符，配置从黑名单优先修改为房间优先。  
之前为：黑名单优先，如果配置了黑名单，那么房间配置为开启也是没有效果的。  
现在修改为：房间配置优先，即如果房间配置了开启，那么即使配置黑名单也是无效的

## Bug fix

1. 修复某些情况下主页“完成后打开文件夹”选项失效
2. 修复主页弹幕预设没有“保存预设”选项
3. 修复webhook中弹幕自适应分辨率bug
4. 修复某些情况下上传预设设置错误的bug
